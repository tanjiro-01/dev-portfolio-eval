# Developer Portfolio Evaluator

Monorepo for a GitHub profile evaluation app with a React frontend and Express backend.

## Workspace Structure

- client: React + Vite frontend
- server: Node.js + Express backend
- docs: project planning and phase tracking

## Local Setup

1. Install dependencies

- From repo root: `npm --prefix client install`
- From repo root: `npm --prefix server install`

2. Configure environment files

- Copy server/.env.example to server/.env
- Copy client/.env.example to client/.env

3. Run development servers

- Frontend: `npm --prefix client run dev`
- Backend: `npm --prefix server run dev`
